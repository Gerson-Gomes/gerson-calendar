# Product Guidelines

These guidelines define the visual, interactive, and prose standards for Gerson Calendar.

## Prose & Tone
- **Minimalist & Concise**: Use short, punchy, and modern language. Avoid unnecessary jargon and keep UI labels clear and direct.

## Visual Style
- **Vibrant Modern Aesthetic**: A deep dark mode theme with vibrant blue and violet accents, modern typography (Nunito), and subtle animations (fadeIn, slideUp) for a premium feel.

## UX Principles
- **Speed & Responsiveness**: Prioritize performance. Every interaction should feel instantaneous.
- **Simplicity & Focus**: Keep the interface clean and clutter-free. Only show essential information and controls.
- **Native Integration**: Use native OS dialogs, file pickers, and notifications to ensure a seamless experience with the host system.

## Design System
- **Custom Components**: Build a bespoke set of UI components that adhere to the project's visual style and UX principles, ensuring a unique and cohesive user experience.
